# Auto Daily Check-in

Automate daily check-ins for multiple games and platforms.  
현재는 `명일방주: 엔드필드(SKPORT)`의 일일 출석 체크를 백그라운드에서 자동으로 수행하는 데 최적화되어 있습니다.

| 항목 | 내용 |
| --- | -------- |
| **플랫폼** | Chrome, Edge, Whale 등 Chromium 기반 브라우저 |
| **버전** | 8.2 (Endfield Optimized & UTC+8 Sync) |
| **필수 조건** | 해당 게임 사이트에 **로그인**이 되어 있어야 함 |

## ✨ 주요 기능

- **자동 출석**: 브라우저를 실행하거나 매 1시간마다 백그라운드에서 자동으로 출석을 확인하고 보상을 수령합니다.
- **서버 시간 동기화 (UTC+8)**: 사용자 PC의 로컬 시간이 아닌, `엔드필드 서버 초기화 시간(00:00 UTC+8)`을 기준으로 하루를 계산하여 정확하게 출석합니다.
- **스마트 감지**: 이미 출석을 완료했거나 미래의 잠긴 날짜인 경우, 이를 지능적으로 파악하여 '완료' 상태로 처리합니다.
- **제어 기능 (Toggle)**
  - **전체 제어**: 확장 프로그램을 끄지 않고도 자동 실행 기능만 켜고 끌 수 있습니다.
  - **사이트별 제어**: 등록된 사이트별로 개별적인 ON/OFF가 가능합니다.
- **시스템 알림**: 크롬 토스트 알림을 통해 작업 성공/실패 여부를 방해 없이 알려줍니다.
- **클라우드 동기화**: Google 계정을 통해 여러 PC에서 설정한 사이트 목록이 공유됩니다.

## 📖 사용 방법

### 1. 사이트 등록 (가장 쉬운 방법)
1. **[명일방주: 엔드필드(SKPORT)](https://www.skport.com/)** 출석 체크 페이지에 접속합니다.
2. 브라우저 우측 상단의 `아이콘(확장 프로그램)`을 클릭합니다.
3. **[+ 현재 보고 있는 사이트 추가]** 버튼을 누릅니다.
4. 자동으로 입력된 이름과 URL을 확인하고 `[저장하기]`를 누릅니다.

### 2. 자동 실행
- 상단의 **'자동 실행'** 토글이 켜져 있다면, 별도의 조작 없이 브라우저 실행 시 자동으로 출석을 시도합니다.
- 출석 성공 여부는 **UTC+8(서버 시간)** 기준으로 갱신되며, 팝업 목록에 `녹색 체크(✅)`가 표시됩니다.

### 3. 수동 실행 (즉시 실행)
- 자동 실행이 꺼져 있어도, 팝업창 상단의 **[즉시 실행]** 버튼을 누르면 강제로 점검을 수행할 수 있습니다.

## ⚠️ 주의 사항 및 FAQ

- **Q. 한국 시간(KST)으로 00시가 지났는데 출석이 안 돼요.**
  - 이 프로그램은 게임 서버 시간인 **UTC+8 00:00 (한국 시간 01:00)**을 기준으로 하루를 계산합니다. 한국 시간 새벽 1시 이후에 자동으로 갱신됩니다.
- **Q. 호요랩(HoYoLAB)은 지원하나요?**
  - 현재 버전(v8.2)부터는 엔드필드(SKPORT) 최적화를 위해 호요랩 기능이 제외되었습니다.
- **Q. 출석이 실패했다고 떠요.**
  - 해당 웹사이트에 **로그인이 되어 있는지** 확인해주세요. 로그인이 풀려있으면 작동하지 않습니다.
- **Q. 알림이 안 떠요.**
  - 운영체제(Windows/Mac)의 '방해 금지 모드'나 크롬 알림 설정이 꺼져있는지 확인해주세요.
