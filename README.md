# Game Auto Check-in Master

게임 웹사이트(호요랩, 명일방주: 엔드필드 등)의 일일 출석 체크를 백그라운드에서 자동으로 수행해주는 크롬 확장 프로그램입니다.

| 항목 | 내용 |
| --- | -------- |
| **플랫폼** | Chrome, Edge, Whale 등 Chromium 기반 브라우저 |
| **버전** | 6.0 (API Hybrid & Google Sync) |
| **필수 조건** | 해당 게임 사이트에 **로그인**이 되어 있어야 함 |

## ✨ 주요 기능

- **자동 출석**: 브라우저를 실행하면 백그라운드에서 자동으로 출석 여부를 확인하고 보상을 수령합니다.
- **API 하이브리드 모드**
  - **HoYoLAB**: API 직접 호출 방식을 사용하여 빠르고 정확하게 처리합니다.
  - **SKPORT**: 화면 요소를 분석하여 자동으로 클릭합니다.
- **클라우드 동기화**: Google 계정을 통해 여러 PC에서 설정한 사이트 목록이 공유됩니다.
- **모던 UI**: Apple iOS 스타일의 깔끔한 디자인과 다크/라이트 모드(시스템)를 지원합니다.
- **스마트 감지**: 이미 출석을 완료했다면 중복 시도를 하지 않고 '완료' 상태로 표시합니다.

## 🚀 설치 방법

이 프로그램은 스토어에 등록되지 않았으므로 **'개발자 모드'**를 통해 수동으로 설치해야 합니다.

1.  **다운로드**: 이 저장소의 `[Code] -> [Download ZIP]`을 클릭하여 다운로드 후 **압축을 풉니다.**
2.  **확장 프로그램 관리 페이지 접속**
    - 크롬 주소창에 `chrome://extensions` 입력 후 엔터 키를 누릅니다.
3.  **개발자 모드 활성화**
    - 우측 상단의 **[개발자 모드]** 스위치를 켭니다.
4.  **확장 프로그램 로드**
    - 좌측 상단 **[압축해제된 확장 프로그램을 로드합니다]** 버튼을 클릭합니다.
    - 압축을 푼 폴더(파일들이 들어있는 폴더)를 선택합니다.
5.  **완료**: 목록에 `Auto Daily Check-in` 카드가 나타나면 설치 완료입니다.

## 📖 사용 방법

### 1. 사이트 등록 (가장 쉬운 방법)
1. 출석 체크를 원하는 웹사이트(예: 호요랩 출석 페이지)에 접속합니다.
2. 브라우저 우측 상단의 `주황색 아이콘(확장 프로그램)`을 클릭합니다.
3. **[+ 현재 보고 있는 사이트 추가]** 버튼을 누릅니다.
4. 자동으로 입력된 이름과 URL을 확인하고 `[저장하기]`를 누릅니다.

### 2. 자동 실행
- 별도의 조작 없이 **브라우저를 켜면** 백그라운드에서 자동으로 출석을 시도합니다.
- 출석이 성공하면 팝업 목록에 `녹색 체크(✅)`가 표시됩니다.

### 3. 수동 실행 (즉시 실행)
- 팝업창 상단의 **[즉시 실행]** 버튼을 누르면 등록된 모든 사이트를 순회하며 점검합니다.

## ⚠️ 주의 사항 및 FAQ

- **Q. 호요랩(HoYoLAB) 등록 시 주의할 점이 있나요?**
  - 반드시 URL 뒤에 `?act_id=...` 파라미터가 붙어있는 주소여야 합니다. 이벤트 페이지에 직접 접속해서 **[현재 사이트 추가]** 버튼을 사용하는 것을 권장합니다.
- **Q. 출석이 안 돼요.**
  - 해당 웹사이트에 **로그인이 풀려있는지** 확인해주세요. 로그인이 되어 있어야 쿠키를 사용하여 출석할 수 있습니다.
- **Q. 삭제는 어떻게 하나요?**
  - 목록의 휴지통 아이콘을 누르면 삭제할 수 있습니다.
